My Favorite CSS Cheat Sheet

CSS Selectors and Specificty

Selector Example Description Specificity Level
Type p Selects all <p> elements Low - least specific and thus easier to modify or change - all instances of an element would be styled
Class .nav Selects all elements with class="nav" Medium - only instances of an element with specific class would be styled
Id #order Selects all elements with id="order" High - overrides type and class, only an element with unique id would be styled

Note- the !important flag can be applied to specific attributes and will override any style regardless of specificty


Common CSS Properties

Property Description Example
font-family Specifies the typeface of an element font-family: Arial
font-size Defines the size of text displayed font-size: 18px
font-weight Specifies thickness of text font-weight: bold
color Defines color of the text color: azure
background-color Defines color behind the text background-color: linen
text-align Specifies alignment of text in an element text-align: center
text-decoration Specifies decoration added to text text-decoration-line: underline
